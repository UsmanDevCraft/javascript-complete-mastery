# ✅ **EASY (Variables — 3 Problems)**

---

### **Problem 1 — Output?**

```js
let a = 10;
a = 20;
console.log(a);
```

### **Solution:**

```
20
```

Let allows reassignment.

---

### **Problem 2 — Output?**

```js
const x = 5;
x = 10;
console.log(x);
```

### **Solution:**

Error:

```
TypeError: Assignment to constant variable
```

`const` cannot be reassigned.

---

### **Problem 3 — Output?**

```js
var a = 1;
var a = 2;
console.log(a);
```

### **Solution:**

```
2
```

`var` allows redeclaration.
