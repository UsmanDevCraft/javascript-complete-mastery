### **1) What will be the output? (Global Scope)**

```js
let x = 10;

function print() {
  console.log(x);
}

print();
console.log(x);
```

### **2) Block Scope with let**

```js
if (true) {
  let a = 5;
}

console.log(a);
```

✔ Will it print `5` or error? Why?
